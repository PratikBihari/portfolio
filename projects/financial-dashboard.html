Project Overview

InvestCorp Financial, a mid-sized investment firm managing over $2 billion in assets, was struggling with fragmented financial reporting systems. Their executive team was spending excessive time gathering and interpreting data from multiple sources, leading to delayed decision-making and missed opportunities.

The company needed a centralized, interactive dashboard that could provide real-time insights into key financial metrics, enabling executives to make data-driven decisions quickly and confidently.

Technical Approach

After a comprehensive assessment of the client's data infrastructure and reporting needs, I designed and implemented a Power BI dashboard solution with the following components:

Data Integration

The first challenge was to integrate data from multiple sources, including:

  • SQL Server database (transaction data)
  • Excel spreadsheets (budget and forecast data)
  • Salesforce CRM (client relationship data)
  • Bloomberg Terminal (market data)

I created a robust ETL (Extract, Transform, Load) pipeline using Power Query to clean, transform, and consolidate these disparate data sources into a unified data model.

Data Modeling

I implemented a star schema data model with fact and dimension tables to optimize query performance and enable complex analytical capabilities. This included:

  • Creating date, client, product, and geography dimension tables
  • Establishing proper relationships between tables
  • Implementing row-level security to ensure appropriate data access
// DAX measure for calculating YoY growth
YoY Growth % = 
VAR CurrentYearSales = CALCULATE(SUM(Sales[Revenue]), 
                        FILTER(ALL(Date), Date[Year] = MAX(Date[Year])))
VAR PreviousYearSales = CALCULATE(SUM(Sales[Revenue]), 
                        FILTER(ALL(Date), Date[Year] = MAX(Date[Year]) - 1))
RETURN
    IF(PreviousYearSales <> 0,
        DIVIDE(CurrentYearSales - PreviousYearSales, PreviousYearSales),
        BLANK())

Dashboard Development

I designed a multi-page dashboard with interactive visualizations that allowed users to:

  • View high-level KPIs with the ability to drill down into detailed analyses
  • Compare actual performance against targets and historical data
  • Analyze trends and patterns across different dimensions (time, geography, product)
  • Perform what-if analyses to simulate different scenarios

Challenges & Solutions

Throughout the project, several challenges emerged that required innovative solutions:

Data Quality Issues

The client's data contained inconsistencies and duplications across different systems. I implemented automated data validation rules and cleansing procedures to ensure data integrity before it entered the dashboard.

Performance Optimization

Initial dashboard prototypes experienced slow refresh rates due to the large volume of data. I optimized performance by:

  • Implementing incremental refresh policies
  • Creating aggregation tables for frequently accessed metrics
  • Optimizing DAX measures and relationships
  • Using composite models to balance import and DirectQuery modes

User Adoption

Some executives were hesitant to transition from their familiar Excel-based reports. To address this, I:

  • Conducted personalized training sessions for key stakeholders
  • Created intuitive, user-friendly interfaces with familiar terminology
  • Implemented Excel export capabilities for users who preferred that format
  • Developed comprehensive documentation and quick reference guides

Conclusion

The financial performance dashboard project transformed how InvestCorp Financial monitored and analyzed their business performance. By providing executives with timely, accurate, and actionable insights, the dashboard enabled more informed decision-making that directly contributed to improved profitability and operational efficiency.

The success of this project led to an expanded engagement with InvestCorp, including the development of additional dashboards for portfolio management, risk assessment, and client relationship management.
